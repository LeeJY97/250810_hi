소개

3년차 Java Spring 개발자로서, 안정적이고 효율적인 백엔드 시스템 개발에 열정을 가지고 있습니다. 클린 코드와 테스트 주도 개발(TDD)을 통해 유지보수하기 좋은 코드를 작성하는 것을 중요하게 생각합니다.

새로운 기술을 배우고 적용하는 것에 즐거움을 느끼며, 동료들과의 협업을 통해 더 나은 결과물을 만들어내는 과정에서 큰 보람을 느낍니다. 현재는 MSA(Microservice Architecture)와 클라우드 네이티브 기술에 깊은 관심을 가지고 학습하고 있습니다.
