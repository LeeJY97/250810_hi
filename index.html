소개

사용자에게 안정적인 서비스를 제공하는 클라우드 인프라 구축에 큰 관심을 가지고 있습니다. 리눅스 환경에 익숙하며, 인프라 자동화와 효율적인 시스템 운영에 대해 꾸준히 학습하고 있습니다.

Java는 아직 익숙하지 않지만, 새로운 기술을 배우는 것에 대한 두려움이 없습니다. 동료들과의 적극적인 소통과 협업을 통해 함께 성장하며, 견고한 서비스를 만들어나가는 과정에서 보람을 느낍니다. 클라우드 네이티브 기술을 깊이 있게 이해하고 적용하는 엔지니어가 되는 것이 저의 목표입니다.

프로젝트

AWS 기반 웹 서비스 배포

EC2, S3, RDS를 활용하여 웹 애플리케이션을 AWS에 배포하고 운영한 경험이 있습니다. CI/CD 파이프라인을 구축하여 배포 자동화를 구현했습니다.

AWS Docker Jenkins
Kubernetes 클러스터 구축 및 운영

On-premise 환경에서 Kubernetes 클러스터를 직접 구축하고, 컨테이너화된 애플리케이션을 배포 및 관리했습니다. 모니터링 시스템을 연동하여 안정성을 높였습니다.

Kubernetes Docker Prometheus
Terraform을 이용한 인프라 코드화(IaC)

Terraform을 사용하여 AWS 인프라를 코드로 관리하고, 반복적인 인프라 생성 작업을 자동화했습니다. Git을 통해 인프라 코드의 형상 관리를 수행했습니다.
